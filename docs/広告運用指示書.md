# カブシグナル 広告運用指示書(Instagram / TikTok 広告用)

このドキュメントは、**別チャットで広告(Instagram広告・TikTok広告)を企画・運用する**ための引き継ぎ資料です。サービスの理解・訴求ポイント・法的制約・クリエイティブ方針をまとめています。

---

## 1. サービス概要

- **サービス名**: カブシグナル(正式: カブシグナル 〈株式市場シグナルウォッチ〉)
- **URL**: https://kabusig.com
- **運営者**: フジワラ / お問い合わせ: admin@kabusig.com
- **一言でいうと**: 東証プライム全銘柄(約1,550銘柄)を、33種類のテクニカル指標・古典的な相場のシグナルで毎日機械的にチェックし、「条件に一致した事実」だけを届ける情報サービス。
- **提供物**:
  - シグナル検知(古典テクニカル17種 + 酒田五法など8種 + 著名分析家由来8種)
  - 検知後1・2・3営業日の値動き実績表示
  - シグナル別の過去統計(発生回数・上昇した割合・平均騰落率)とバックテスト
  - 経済ニュースまとめ(公式RSS集約)
  - LINE通知(監視銘柄 or シグナル単位で全銘柄)
  - 相場の暦・アノマリーカレンダー

## 2. 料金・課金モデル

- **プレミアム会員**: 月額 **980円(税込)** / クレジットカード(Stripe)/ 毎月自動更新
- **無料プランは無し**。ただし**未登録でも一部は閲覧可能**(下記「餌」設計)。
- 解約: アカウント画面の「お支払い管理ポータル」からいつでも可能。

## 3. 「餌」設計(無料→有料の動線)= 広告の肝

広告で連れてきた人を、無料で惹きつけ→プレミアムに誘導する導線が組んである。広告クリエイティブはこの「見えるところ」を訴求する。

| コンテンツ | 未登録/無料で見える範囲 | プレミアムで見える範囲 |
|---|---|---|
| 経済ニュースまとめ | **全部見える**(毎日ログインの動機) | 同じ |
| シグナル検知(当日) | 10件まで | 全件 |
| 検知後の結果 | **1営業日後だけ**見える | 2・3営業日後も |
| 上昇割合ランキング | **1位だけ**見える | 全順位・数値 |
| 平均騰落率ランキング | **1位だけ**見える | 全順位・数値 |
| シグナル統計・バックテスト | ロック | 全部 |
| 銘柄詳細・LINE通知・監視銘柄 | ロック | 全部 |

→ **広告の訴求例**: 「毎日の経済ニュースを無料でチェック」「シグナルが出たあと株価がどう動いたか、実績で見える」など。ニュース無料閲覧を入口にするのが摩擦が少ない。

## 4. ターゲット層

- 個人投資家(初〜中級)、日本株(東証プライム)に関心がある層
- チャート/テクニカル指標に興味はあるが、自分で毎日全銘柄をチェックするのは大変な層
- 「感覚ではなく事実・データで相場を見たい」層(サービスのトーンは"事実で見る")
- 年齢: 30〜60代想定。Instagramはやや上、TikTokは若年〜中年で認知拡大。

## 5. ブランド・トーン

- キーメッセージ: **「市場を 事実で見る。」**
- デザイン: Apple風のミニマル・高級感・白基調・余白広め
- ロゴ/favicon: メタリックな上向き矢印
- 声のトーン: 冷静・誠実・煽らない。「儲かる!」ではなく「事実を届ける」。

## 6. 【最重要】法的制約 — 広告にも必ず適用

このサービスは**投資助言・投資勧誘に該当してはいけない**(金融商品取引法)。広告表現も同じ制約を受ける。加えて景品表示法(誇大広告禁止)にも注意。

### 絶対に使ってはいけない表現(広告文・動画・サムネ含む)
- 特定銘柄の売買を勧める表現全般
- 「儲かる」「必ず勝てる」「必勝」「爆上げ」「急騰確実」など利益を保証・示唆する表現
- 「買い時」「売り時」「買うべき」「おすすめ銘柄」「注目銘柄」など投資判断を助言する表現
- 「元本保証」「絶対」「確実」など断定的表現
- ※厳密な禁止語リストはリポジトリの `batch/tests/test_forbidden_words.py` を参照(サービス本体は全ソースを機械チェックしている)

### 使ってよい方向性(事実ベース)
- 「東証プライム全銘柄を毎日自動チェック」(機能の事実)
- 「シグナル検知後、実際にどう動いたかを実績で表示」(過去の事実)
- 「〇〇というシグナルの過去の上昇割合は△△%(過去統計)」→ 必ず**「過去の統計であり将来を保証しない」旨**を併記
- 「毎日の経済ニュースを1か所で」(機能)
- 「情報ツールであり、投資助言ではありません」の免責を必ず添える

### 広告に必ず入れる免責(小さくてもよいが必須)
> 本サービスは情報提供ツールであり、投資勧誘・投資助言を目的とするものではありません。投資判断はご自身の責任で行ってください。

## 7. クリエイティブ方針(媒体別)

### Instagram(フィード/リール/ストーリーズ)
- ビジュアル: 白基調・データUIのスクショ(統計ランキング・チャート・ニュース一覧)
- フック例: 「その株、シグナルが出たあとどう動いた?」「毎朝の相場チェックを1画面で」
- CTA: 「無料でニュースを見る」→ kabusig.com

### TikTok(短尺動画)
- 15〜30秒。画面録画でUIを見せる+テロップ。BGMは流行りの落ち着いたもの。
- フック(最初の2秒)が命: 「株の"事実"だけ見たい人へ」など
- 煽り禁止(法的制約)。淡々と機能を見せるのが逆に信頼感。
- CTA: プロフィールリンク → kabusig.com

## 8. 計測・KPI

- サイトには **Vercel Web Analytics** 導入済み(Cookie不使用)。Vercel管理画面の「Analytics」タブで訪問者数・流入元・人気ページ・国・端末が見られる。
- **UTMパラメータ**を広告リンクに付けると流入元を分離できる(例: `https://kabusig.com/?utm_source=instagram&utm_medium=cpc&utm_campaign=launch`)。
- 主要KPI案: サイト訪問数 → 会員登録数(無料マジックリンク登録)→ プレミアム課金率(980円決済)。
- 課金はStripe(本番)で計測。Stripeダッシュボードで売上・サブスク数を確認。

## 9. ランディング先の候補

- トップ `/`(ニュース+シグナル+ランキングの総合。無料で一番"餌"が効く)
- 料金/紹介 `/pricing`(プレミアムで何が見られるか)
- 統計 `/stats` は会員限定(未登録はペイウォール)なので広告LPには不向き

## 10. 参考: 競合との差別化

- 「特定銘柄を推奨するサービス」ではなく「**事実・統計を淡々と出す中立ツール**」という立ち位置が差別化かつ法的安全性の源泉。広告でもこの中立性を崩さない。
